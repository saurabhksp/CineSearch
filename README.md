
# CineSearch

CineSearch is a React-based movie finder web app that allows users to search movies, view detailed information, and manage a favorites list.

## Features
- Search movies by name
- View movie details (poster, year, plot, rating, etc.)
- Add and remove favorites
- Favorites saved using localStorage
- Pagination for browsing results
- Filter by type (movie/series/episode)
- Sorting by title and year

## Tech Stack
- React (Create React App)
- JavaScript (ES6+)
- HTML5
- CSS3
